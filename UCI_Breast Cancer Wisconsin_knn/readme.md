## KNN Classification with PCA

This exercise implements a **K-Nearest Neighbors (KNN) classification pipeline** using Scikit-Learn. The workflow includes **data preprocessing, handling missing values with median imputation, feature scaling using StandardScaler, and model training**.

Hyperparameter tuning is performed using **GridSearchCV**, and **Principal Component Analysis (PCA)** is applied to explore dimensionality reduction and its impact on model performance.
